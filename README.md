Todo & Calendar Bot

This Telegram bot helps you manage daily tasks and visualize your schedule.
Features

1. Task Management (To-Do List)
    * Create a new task list for any date
    * Add tasks to an existing list
    * Mark one or multiple tasks as completed (strikethrough)
    * View today's tasks
2. Calendar Visualization
    * Generate and view calendar as an image in three formats:
        * Day — detailed view of one specific day
        * Month — full monthly overview
        * Custom range — any period you choose
    * Book a time slot — add tasks/events with time (e.g. "11 meeting", "17 dentist")
    * Share calendar with another person in three formats (same as view modes):
        * Day / Month / Custom range
        * Three privacy modes:
            * Show all tasks
            * Hide all tasks (empty slots only)
            * Hybrid (selective visibility)
        * Send directly to any Telegram user by their @username through the bot
The bot generates clean, readable calendar images with your tasks placed in the correct time slots.
Built with aiogram 3.x · Python · Pillow (image generation)
